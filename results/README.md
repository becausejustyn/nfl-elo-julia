# results/

This directory is populated by the simulation scripts. All files here are generated and are excluded from version control by default.

| File                         | Generated by          | Description                                  |
|------------------------------|-----------------------|----------------------------------------------|
| `elo_ratings_current.csv`    | `scripts/main.jl`     | Final Elo rating for every team              |
| `elo_game_predictions.csv`   | `scripts/main.jl`     | Full game history with pre-game Elo + prob   |
| `playoff_simulation.csv`     | `scripts/run_playoffs.jl` | SB win % for each playoff team           |
| `season_simulation.csv`      | `scripts/run_season_sim.jl` | Expected wins per team for a schedule  |
| `current_ratings.png`        | `scripts/main.jl --plot` | Bar chart of current Elo ratings          |
| `playoff_odds.png`           | `scripts/run_playoffs.jl` | Bar chart of Super Bowl odds             |
